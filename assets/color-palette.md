## Tone Mapping

| Purpose | Color |
|---|---|
| Primary action | accent |
| Positive / on track | green |
| Warning | yellow |
| Danger / deficit | red |
| Portfolio / FIRE | purple |
| Health / reserve | teal |

## UI Rule

Không đổi sang theme xanh corporate hoặc dark trading terminal nếu chưa có quyết định sản phẩm mới.
