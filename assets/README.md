## Gợi ý cấu trúc

```txt
assets/
```

## Quy tắc

- Không lưu dữ liệu tài chính thật ở đây.
- Không lưu API key.
- Không lưu ảnh nhạy cảm.
- Chỉ lưu guideline, mockup, icon notes, chart references.
